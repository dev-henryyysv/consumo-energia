# ⚡ Calculadora de Consumo de Energia

## 📌 Objetivo
Calcular o consumo mensal de energia de um aparelho.

## 🧠 Como funciona
O usuário informa:
- Nome do aparelho
- Potência (W)
- Horas por dia

## 📊 Fórmula
Consumo = (potência × horas × 30) / 1000

## 💰 Custo
Baseado em R$ 0,75 por kWh

## ▶️ Como executar
Abra o app.py no Pydroid e clique em Run.
